# Дашборд статистики Башкортостан

Проект сброшен к чистому листу. Дальше — новая архитектура и интерфейс с нуля.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://127.0.0.1:5173

Опционально API-заготовка:

```bash
npm run dev:api
```

http://127.0.0.1:4100/api/health

## Проверка

```bash
npm test
npm run build
```
