## Что подходит именно для дашборда «рейтинг 63 районов»

### Уровень A — уже использовать (ядро)

1. **tochno.st BDMO** — сотни показателей, все МО, официальная база.  
2. **opendata.sf2.simai.ru** — оперативные реестры и ведомственные CSV.  
3. **Архив BDMO 2003–2023** — через облачные JSON (адаптер в `archiveData.ts`).

### Уровень B — подключить следующим

4. **Разделы BDMO 31–32 и 13** (население, занятость, местный бюджет) — большие архивы на tochno.st.  
5. **tochno.st budget_expenses** — ежемесячный бюджет республики (контекст для руководства).  
6. **Fedstat** — 20–50 ключевых показателей с сохранёнными SDMX-spec (республика + динамика).  
7. **02.rosstat.gov.ru** — парсинг квартальных бюллетеней с таблицами по МР/ГО.

### Уровень C — низкий приоритет

- statbase.ru, mojgorod.ru (нет 63 МО)  
- bus.gov.ru (учреждения, не сводная статистика)  
- 63 отдельных сайта «открытый бюджет»  
- ГАС «Управление» без согласования доступа  
- Платный API ГМЦ Росстата
